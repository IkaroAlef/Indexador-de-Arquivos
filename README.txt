FUNCIONAL: INDEXADOR DE ARQUIVOS

------------
Passos para uso:
1 - Instalar o Plugin da linguagem Scala no Eclipse, pode ser feito pelo Marketplace do Eclipse
https://marketplace.eclipse.org/content/scala-ide

2 - Adicionar todos os arquivos arquivos JARs que estão na pasta Lib ao Classpath do projeto antes do momento de compilação.
Obs.: o arquivo .classpath atual já possuem as Libs importadas, em qualquer caso é bom certificar de que as Libs estão importadas corretamente.

3 - Ao executar o build do arquivo Searcher.scala, o projeto será compilado e estará pronto para uso.
Main: Searcher.scala
